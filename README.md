# 折腾笔记

```javascript
1.转换bv或av号获取视频Cid
https://api.bilibili.com/x/player/pagelist?bvid=BV号
2.通过BV号以及Cid获取视频真实播放地址
https://api.bilibili.com/x/player/playurl?bvid=BV号&cid=cid号&qn=qn号&type=&otype=json&platform=html5&high_quality=1

"超清 1080P+",116,
"高清 1080P",80,
